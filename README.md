# Neovim Configuration

## Installation

```sh
cd ~
git clone git@github.com:degete/nvim-config.git ~/.nvim
ln -s ~/.nvim .config/nvim
```

### Plugins

To update the plugins, you can just open nvim or run the following command:

```sh
nvim --headless "+Lazy! sync" +qa
```
